# Contributing

Thanks for your interest in contributing to **pxe-server**!

## Getting Started

1. Fork the repository and clone your fork.
2. Create a feature branch: `git checkout -b my-feature`.
3. Make your changes, keeping commits small and focused.
4. Push to your fork and open a Pull Request against `main`.

## Guidelines

- Follow the existing code style (see `.editorconfig`).
- Write clear commit messages that explain *why*, not just *what*.
- Keep pull requests focused on a single concern.
- Be respectful — see our [Code of Conduct](CODE_OF_CONDUCT.md).

## Reporting Issues

Open a GitHub issue with a clear description, steps to reproduce, and any
relevant logs or configuration snippets (redact secrets).

## Security

If you discover a security vulnerability, please report it privately.
See [SECURITY.md](SECURITY.md) for details.
